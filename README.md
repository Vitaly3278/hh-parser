# Трекер вакансий hh.ru с уведомлениями в Telegram

Приложение автоматически отслеживает новые вакансии на hh.ru и отправляет уведомления в Telegram.

## Структура проекта

```
├── main.py           # Главный скрипт запуска
├── hh_parser.py      # Модуль работы с API hh.ru
├── telegram_bot.py   # Модуль отправки уведомлений
├── database.py       # Модуль базы данных
├── config.py         # Файл конфигурации
├── requirements.txt  # Зависимости
└── README.md         # Документация
```

## Установка

1. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

2. **Настройте Telegram бота:**
   - Создайте бота через [@BotFather](https://t.me/BotFather)
   - Получите токен бота
   - Узнайте свой chat_id через [@userinfobot](https://t.me/userinfobot)
   - Напишите боту любое сообщение

3. **Отредактируйте `config.py`:**
   ```python
   TELEGRAM_BOT_TOKEN = "ваш_токен"
   TELEGRAM_CHAT_ID = "ваш_chat_id"
   HH_SEARCH_TEXT = "Python разработчик"  # Ваш запрос
   HH_AREA = "1"  # 1=Москва, 2=СПб, None=все регионы
   CHECK_INTERVAL = 300  # Интервал проверки (сек)
   ```

## Запуск

```bash
python main.py
```

## Параметры поиска (config.py)

| Параметр | Описание | Пример |
|----------|----------|--------|
| `HH_SEARCH_TEXT` | Поисковый запрос | "Python разработчик" |
| `HH_AREA` | Регион | "1"=Москва, "2"=СПб, "104"=Россия |
| `HH_SALARY_FROM` | Мин. зарплата | 100000 или None |
| `HH_EMPLOYMENT` | Тип занятости | ["full", "part"] или None |
| `HH_EXPERIENCE` | Опыт работы | ["noExperience", "between1And3"] или None |
| `CHECK_INTERVAL` | Интервал проверки | 300 (5 минут) |

## Коды регионов

- `1` — Москва
- `2` — Санкт-Петербург
- `104` — Россия (все регионы)
- `3` — Екатеринбург
- `88` — Казань
- `1104` — Новосибирск

[Полный список регионов](https://github.com/hhru/api/blob/master/docs/areas.md)

## Как это работает

1. Приложение запрашивает вакансии с hh.ru API
2. Сравнивает с базой данных (SQLite)
3. Новые вакансии отправляет в Telegram
4. Старые записи (>30 дней) удаляются автоматически
